# 🚨 Crime Clock

An interactive “live” crime counter using UK crime statistics.

## 📊 Data Sources
- Office for National Statistics (ONS): ~9.6 million crimes/year
- Police.uk API: https://data.police.uk/

## ⚙️ Features
- Real-time animated counter
- Based on real UK crime rates
- Optional integration with Police API
- Lightweight (no frameworks)

## 🚀 How to Use
1. Clone repo
2. Open `index.html`
3. Done ✅

## 💡 Use Case
Designed for:
- UCAS fairs
- Interactive displays
- Recruitment for policing & criminal investigation courses

## ⚠️ Note
This is a **simulated live counter** based on real statistics.
